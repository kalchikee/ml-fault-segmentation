Methodology

1. Catalog preparation
Download M≥1.0 events with horizontal uncertainty <1 km. Project to UTM Zone 11N for metric clustering.

2. HDBSCAN clustering
3D feature matrix [x, y, depth×2]. Optimal min_cluster_size=100 via silhouette sweep.

3. PCA fault planes
SVD of each cluster → third eigenvector = fault normal → strike & dip.

4. Comparison
Detected traces vs USGS Quaternary Fault Database. UNMATCHED clusters = candidate unmapped structures.
